# Project A: Interactive Risk Dashboard

## Overview
Streamlit multi-page dashboard for real-time risk monitoring and player management.

## Features
- Player risk scoring with ML
- Multi-account detection
- Visual KPI tracking
- CSV export for action

## Files
- `risk_dashboard.py` - Main dashboard
- `screenshots/` - UI screenshots
- `sample_output/` - Example exports

## Run
```bash
streamlit run risk_dashboard.py
```

## Tech
Python, Streamlit, pandas, plotly
