# Project B: ML Anomaly Detection

## Overview
Machine learning system for automated bot detection and risk scoring.

## Key Achievement
**457 bot accounts identified** through same-second timestamp analysis.

## Methods
1. Isolation Forest (unsupervised ML)
2. Z-Score statistical analysis
3. IQR outlier detection
4. Timestamp pattern analysis

## Files
- `ml_anomaly.py` - ML detection
- `sample_output/ml_anomaly_detection.xlsx` - Results

## Tech
Python, scikit-learn, scipy, pandas
