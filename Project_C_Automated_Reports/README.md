# Project C: Automated Reports

## Overview
Professional PDF report generation system with daily/weekly automation.

## Reports
1. **Daily Risk Report** - Top risks, bot suspects, sport breakdown
2. **Weekly Executive Summary** - Trends, recommendations, loss analysis

## Files
- `daily_report.py` - Daily report generator
- `weekly_report.py` - Weekly summary
- `sample_output/*.pdf` - Example reports

## Features
- Automated chart generation
- Executive recommendations
- Scheduling-ready (Windows Task Scheduler)

## Tech
Python, matplotlib, pandas, reportlab
